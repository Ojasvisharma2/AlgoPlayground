# AlgoPlayground (C++)

A command-line tool implementing multiple Data Structures and Algorithms.

## Features
- Kadane's Algorithm (Max Subarray)
- Two Pointers (Pair Sum)
- Sliding Window
- Binary Search
- Greedy (Coin Change)
- Dynamic Programming (Climbing Stairs)
- Recursion (Subset Sum)

## How to Run
1. Compile:
   g++ main.cpp -o algo

2. Run:
   ./algo

## Concepts Used
- Arrays & Vectors
- Sorting
- Recursion
- Greedy & DP
- Sliding Window
- Binary Search

## Purpose
To practice and demonstrate core DSA concepts in a single interactive CLI project.
